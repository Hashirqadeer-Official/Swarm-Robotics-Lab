# Demo 01 - Hello World Serial

## Objective
Print a startup message from ESP32-S3 to Serial Monitor.

## Board
ESP32-S3 DevKitC-1

## Baud Rate
115200

## Files
- `src/main.cpp`
- `platformio.ini`
- `wokwi.toml`
- `diagram.json`

## Steps
1. Build the project using PlatformIO.
2. Open Wokwi simulation.
3. Start the simulation.
4. Open Serial Monitor at 115200 baud.
5. Verify the output.

```text
